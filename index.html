About RecipePal

RecipePal is a native Android application built with Kotlin and Jetpack Compose. It helps users browse recipes, select dietary preferences, search by ingredient or country, save favorites, and view detailed cooking instructions.

Features

🥗 Diet Filtering

Browse vegetarian, vegan, pescetarian, kosher, heritage, and other recipe categories.

🔎 Recipe Search

Search recipes by name, country, dietary category, or ingredient.

❤️ Favorites

Save preferred recipes and quickly access them from the Favorites screen.

📖 Recipe Details

View ingredients, cooking steps, preparation time, servings, and allergens.
